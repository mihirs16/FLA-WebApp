sand. Simulation App for NFA & DFA
Deterministic Finite Automata DFA refers to Deterministic Finite Automaton. A Finite Automata (FA) is said to be deterministic, if corresponding to an input symbol, there is single resultant state i.e. there is only one transition.
Non-Deterministic Finite Automata NFA refers to Nondeterministic Finite Automaton. A Finite Automata (FA) is said to be non deterministic, if there is more than one possible transition from one state on the same input symbol.
